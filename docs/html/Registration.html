Registration¶

Most of the applications require account to collect payments or restrict usage.

Username-Password¶

The most basic way is to use username-password combination, and having email assigned to account to restore lost password. However, because of email delivery issues it might be better to use SMS instead.

app.user.create(username: str, password: str) → dict¶

Creates a new user if the username is not already used.

Parameters

  • username – The name to use

  • password – Password in plain-text

Returns

created user as { id, username }

app.user.create_password(plain_password: str) → str¶

Returns a password hash.

Social Media¶

Another common registration method today is to register via a large social media account such as Google or Facebook.
